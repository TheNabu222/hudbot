🚀 HOW TO USE - 3 EASY WAYS
═══════════════════════════════════════════════════════════════════

METHOD 1: OPEN LOCALLY (Easiest!)
──────────────────────────────────
1. Find the "index.html" file in this folder
2. Double-click it to open in your web browser
3. Start creating!

METHOD 2: USE A WEB SERVER (Recommended)
─────────────────────────────────────────
If you have Python installed:
1. Open Terminal/Command Prompt in this folder
2. Type: python3 -m http.server 8080
3. Open your browser and go to: http://localhost:8080

If you have Node.js installed:
1. Open Terminal/Command Prompt in this folder
2. Type: npx serve .
3. Follow the URL shown in the terminal

METHOD 3: DEPLOY ONLINE (Share with the world!)
────────────────────────────────────────────────
Upload all files in this folder to any of these services:
• Neocities (https://neocities.org) - Perfect for static sites!
• GitHub Pages - Free hosting with Git
• Netlify (https://netlify.com) - Drag & drop deployment
• Vercel (https://vercel.com) - One-click deployment

💡 GETTING STARTED
═══════════════════════════════════════════════════════════════════

Once you open CoAIexist Studio:

1. EXPLORE THE INTERFACE
   • Left panel: Add components and elements
   • Center: Your page preview and editor
   • Right panel: Style and properties

2. CREATE YOUR FIRST PAGE
   • Click "Add Component" to add elements
   • Drag and resize elements visually
   • Use the AI assistant to generate content

3. CUSTOMIZE STYLES
   • Choose from preset themes
   • Modify colors, fonts, and layouts
   • Apply world region aesthetic tags

4. DEPLOY YOUR PROJECT
   • Export your pages
   • Deploy to Neocities directly from the app
   • Or download and upload to any hosting service

🔧 TROUBLESHOOTING
═══════════════════════════════════════════════════════════════════

Problem: Page looks broken or doesn't load properly
Solution: Make sure you're using a modern browser and all files
         are in the same folder

Problem: AI features not working
Solution: You'll need to configure your Gemini AI API key in the
         app settings (look for the settings icon)

Problem: Can't save projects
Solution: Your browser needs to allow localStorage. Check your
         browser privacy settings.
